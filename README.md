Esse trabalho possui 2 códigos o ProdIntSeq.c e ProdIntConc.c sendo o primeiro relacionado ao produto interno calculado de forma sequencial e o segundo de forma concorrente.O primeiro código recebe como entrada o tamanho dos vetores e o nome do arquivo em que será escrito as saídas, nessa ordem, ele então gera vetores de float com o tamanho fornecido na entrada calcula o produto interno e escrever no arquivo em binario a dimensão dos vetores, os vetores e seus valores, e por fim o resultado do produto interno. Já o código concorrente voce deve fornecer como entrada na chamada do executavel o numero de threads que deseja para o código e o nome do arquivo com os dados salvos anteriormente, nessa ordem, assim ele irá calcular o produto interno e retornar no arquivo o valor calculado pelo código junto com as demais informações, ao final o código printa o resultado de |(vs-vc)/vs|.
